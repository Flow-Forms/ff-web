# Display Options

The Control Panel includes clickable icons that allow you to switch between two display options and change the sort order of your displayed submissions. These are found on the right side of the Control Panel below the **{{icon:adjustments-horizontal size-4 inline-block align-text-center}} Filters** button and beside the **{{icon:check-badge size-4 inline-block align-text-center}} Status** quick-filter button. *(Directly below **Clear Filters** when a filter is active.)*


## {{icon:bars-arrow-down size-6 inline-block align-text-center}} Sort Submissions 

Click the {{icon:bars-arrow-down size-6 inline-block align-text-center}} button and your **Sort Submissions** options will appear in a dropdown. The default or active settings are indicated by a checkmark. Click any of the other options to select them instead.

### Sort By

- **Updated Date (Default)** - This sort order displays your submissions according to the date they last had a status update. A status changes when an action is taken on that submission such as Approve, Deny, Request Info, etc.
- **Created Date** - Choose to order your submissions by the date the submission was initiated.

### Direction

- **Newest First (Default) -** Display submissions with the most recent listed first, then in descending order to the oldest.

- **Oldest First -** View submissions in reverse order, with the oldest submission at the top and newest last.

## {{icon:arrows-pointing-out size-6 inline-block align-text-center}} Expand All 

Fully display all of the submissions' data including an expanded view of attachments in some formats (png, jpeg). 

## {{icon:arrows-pointing-in size-6 inline-block align-text-center}} Collapse All 

Display an overview of the submission's elements that have been selected as pertinent at the time of the form's creation. This view option displays attachments as thumbnails and/or viewing links. 


***Note** - You can change the view of an individual submission rather than apply that view to the whole page. Expand an individual submission that is collapsed by clicking the **{{icon:arrows-pointing-out size-6 inline-block align-text-center}} show more** button in that submission's header and, conversely, collapse an expanded submission by clicking its **show less{{icon:arrows-pointing-in size-6 inline-block align-text-center}}** button. 
Additionally, see [Show More](submissions/submission_view#ShowMore) and [Show Less](submissions/submission_view#ShowLess) in the Submission View section.*
